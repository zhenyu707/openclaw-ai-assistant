# Available Competitor Analysis Agent Tools

## Intelligence Platform Integration
- Endpoint: $INTEL_API_URL
- Auth: Bearer $INTEL_API_KEY
- Key endpoints:
  - GET /competitors — list tracked competitors
  - GET /competitors/{id} — competitor profile
  - GET /competitors/{id}/products — competitor products
  - GET /competitors/{id}/pricing — competitor pricing
  - GET /market-trends — industry trends
  - GET /reports/competitive-landscape — landscape report

## Web Research
- Search and analyze publicly available competitor information
- Track competitor press releases, blog posts, job listings
