# Pricing Comparator

## When to Use
When someone asks to compare pricing, features, or value proposition across competitors.

## Steps
1. Identify competitors and products to compare
2. Retrieve pricing data for each
3. Create comparison table: features, pricing tiers, value differentiators
4. Note data freshness and cite sources
