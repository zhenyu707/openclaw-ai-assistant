# HR Policy Q&A

## When to Use
When employee asks about company policies: vacation, leave, code of conduct,
benefits, remote work, expenses, holidays, etc.

## Policy Document Locations
- Leave policy: policies/leave-policy.md
- Benefits: policies/benefits.md
- Code of conduct: policies/code-of-conduct.md
- Remote work: policies/remote-work-policy.md
- Expense: policies/expense-policy.md

## Steps

1. Identify the policy topic from the employee's question
2. Use the read tool to load the relevant policy document
3. Extract the specific relevant section
4. Provide a clear, concise answer citing the policy
5. Offer to escalate to HR Manager if the question needs interpretation

## Important
- Always cite which policy document you're quoting from
- Do NOT interpret policies in ways that change their meaning
- When in doubt, say: "I'd recommend confirming this with the HR Manager"
- If a policy document doesn't exist yet, say: "That policy document is not yet available. Please contact HR directly."
